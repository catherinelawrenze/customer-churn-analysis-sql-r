# 📊 Customer Churn Analysis using SQL and R

> An end-to-end Data Analytics and Predictive Modeling project that analyzes customer churn patterns using SQL and R, and builds a Logistic Regression model to identify customers at risk of leaving.

---

## 📌 Project Overview

Customer churn is one of the biggest challenges faced by telecommunication companies, as losing existing customers directly impacts revenue and long-term business growth.

This project analyzes customer behavior using SQL for business analysis and R for exploratory data analysis, visualization, and predictive modeling. The objective is to identify the key drivers of churn and provide actionable business recommendations for improving customer retention.

---

## 🎯 Objectives

- Calculate the overall customer churn rate.
- Identify high-risk customer segments.
- Analyze the impact of contract type, tenure, payment methods, and internet service on churn.
- Build a Logistic Regression model to predict customer churn.
- Evaluate the predictive model using standard machine learning metrics.

## 📈 Key Findings

- Customers with **Month-to-Month contracts** showed the highest churn rate.
- **Electronic Check** users were significantly more likely to churn.
- **Fiber Optic** customers experienced higher churn compared to DSL users.
- Customers with **short tenure** had a substantially higher probability of leaving.
- Churned customers represented approximately **$139,130.85** in lost monthly revenue.
